# Tikslų sistema

Interaktyvi tikslų siekimo lenta su taikinio principu.

## Kaip veikia

- Žalia zona: veiksmai, kurie veda prie tikslo.
- Geltona zona: veiksmai, kurie akivaizdžiai neprisideda, bet ir netolina nuo tikslo.
- Raudona zona: veiksmai, kurie tolina nuo tikslo.

Sistema leidžia:

- pridėti žodžius į žalią, geltoną arba raudoną zoną;
- redaguoti ir perkelti žodžius;
- automatiškai keisti spalvų plotą pagal žodžių kiekį;
- didinti arba mažinti pasirinktą žodį;
- priartinti / patolinti taikinį pelės ratuku;
- slinkti vaizdą laikant dešinį pelės klavišą;
- rašyti tikslą, dienos planą, motyvaciją ir stabdžius;
- išsaugoti įrašus naršyklėje per `localStorage`.

## Failai

- `index.html` – pagrindinis puslapis.
- `styles.css` – vaizdo stiliai.
- `app.js` – interaktyvi logika.
